# rn-kill

Exit / Close / Kill / shutdown your react native app. Does not invoke a crash notification.

## Installation

```sh
npm install rn-kill
```

## Usage

```js
import { multiply } from 'rn-kill';

// ...

const result = await multiply(3, 7);
```
