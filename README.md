# rn-kill :bomb:

Exit / Close / Kill / shutdown your react native app. Does not invoke a crash notification.

## Installation

```sh
npm install rn-kill
```

## Usage

```js
import RNKill from 'rn-kill';

// ...

RNKill.exitApp();
```
